# Assignment 1
This is source code of my first assignment work from Algorithms and Data Structures. It have 10 functions to 10 tasks.

## Usage
```java
// finding minimum number from array nums
findMin(n, nums);

// finding average from array nums
findAvg(n, nums);

// returns true if number is prime
isDigitPrime(n);

// find factorial of given number
findFactorial(n);

// find number in Fibonacci sequence
findFibonacci(n);

// function that find a power of n
findPower(a, n);

// returning string that contains sequence of array in reverse order
getReverse(n, nums);

// returns true if given string contains all digits
isAllDigit(line);

// finding binomial coefficent by two numbers
findBinom(n, k);

// finding gcd of two numbers
findGCD(a, b);
```
